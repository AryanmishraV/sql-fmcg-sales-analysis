# SQL FMCG Sales Analysis

## Objective
This repository demonstrates the use of SQL to analyze FMCG sales data and answer
real-world business questions related to revenue, products, regions, and growth trends.

The focus is on writing clean, efficient SQL queries that support business
and operational decision-making.

## Skills Demonstrated
- SQL Joins and Aggregations
- GROUP BY, HAVING, ORDER BY
- Subqueries
- Window Functions
- Business KPI analysis

## Business Questions Covered
- Which products generate the highest revenue?
- What are the top and bottom performing regions?
- How does monthly sales trend over time?
- Which categories contribute most to total sales?
- Who are the highest value customers?
- How does sales performance vary by region and category?

## Dataset
The dataset represents typical FMCG sales data including:
- Order date
- Product
- Category
- Region
- Customer
- Quantity
- Revenue
